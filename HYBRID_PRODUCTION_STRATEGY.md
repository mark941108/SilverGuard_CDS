### **步驟 1: 真實螢幕錄影 (用 OBS Studio, 免費)**

#### **準備工作**
1. 下載 OBS Studio: https://obsproject.com/
2. 啟動 `app.py`: `python app.py`
3. OBS 設定:
   - Source: "Window Capture" → 選擇瀏覽器視窗
   - Resolution: 1920x1080
   - FPS: 30

#### **錄影清單**

**Recording 1: Scene 2 - Edge AI Demo** (0:20-0:35)
```bash
# 執行步驟:
1. OBS 開始錄影
2. 上傳一張藥袋圖片到 Gradio
3. 系統正常辨識
4. [手動] 拔掉網路線 或 在瀏覽器 DevTools 設定 "Offline"
5. 介面變灰,顯示 "OFFLINE MODE"
6. OBS 停止錄影
7. 保存為: rec_scene2_offline.mp4
```

**💡 技巧**: 
- 用 Chrome DevTools (F12) → Network → 勾選 "Offline" 模擬斷線

## 🎯 進階優化技巧

### **技巧 1: 一張圖生成多個角度 (省 Credits)**
```
生成 1 張高品質圖 (20 Credits)
→ CapCut 裁切成 3 個不同構圖
→ 等於 3 個鏡頭 (省 40 Credits)
```

### **技巧 2: Veo "Extend" 功能延長時間**
```
生成 8 秒影片 (20 Credits)
→ 用 Flow "Extend Video" 延長到 16 秒 (+10 Credits)
→ 比生成 2 個 8 秒便宜
```

### **技巧 3: 降低 FPS 營造「電影感」**
```
OBS 錄影時設定 24 FPS (而非 30 FPS)
→ 自動產生「Film Look」
→ 不需要後期調色
```

### **技巧 4: 用「黑場」過渡節省片段**
```
Scene 3 → Scene 4 之間插入 1 秒黑場
→ 不需要生成過渡片段
→ 還能增加戲劇張力
```
